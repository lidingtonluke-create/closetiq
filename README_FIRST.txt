CLOSETIQ - EASY START

1. Install Node.js LTS from https://nodejs.org if you do not already have it.
2. Unzip this folder.
3. Windows: double-click START_WINDOWS.bat
   Mac: double-click START_MAC.command
4. The first time, it creates a .env file.
5. Open .env and paste your API keys:
   OPENAI_API_KEY=your_key_here
   REMOVE_BG_API_KEY=your_key_here
6. Double-click the start file again.
7. Click the localhost link that appears in the terminal.

If you do not want AI/background removal yet, the app can still save clothes and generate outfits manually after running.
